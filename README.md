Base de Conhecimento de Jogos FIFA – Busca Inteligente com UI Inspirada em EA Sports

Este projeto é uma aplicação web desenvolvida para oferecer uma experiência moderna, fluida e intuitiva ao pesquisar jogos da franquia FIFA.
Com foco em design, desempenho e usabilidade, a interface foi construída com uma estética inspirada no estilo visual dos menus de EA Sports FC, aplicando elementos como glow suave, paleta personalizada e componentes interativos.

O objetivo é entregar uma ferramenta prática, rápida e elegante, apresentando informações detalhadas dos títulos FIFA, incluindo descrição, ano de lançamento e capa oficial.

Funcionalidades:
- Busca inteligente (filtragem por nome e descrição)
- Cards organizados com informações de cada jogo
- Capa do FIFA exibida automaticamente
- UI moderna com paleta verde personalizada
- Layout responsivo para mobile e desktop
- Interações suaves inspiradas no estilo EA Sports FC
